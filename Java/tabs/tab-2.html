Data Types!

There are two types of data types. First, there are primative data types. Below are some primative data types. 1. byte: It stores 1 byte. 2. short: It stores 2 bytes. 3. int: It stores 4 bytes. 4. long: It stores 8 bytes. 5. float: It stores 4 bytes. 6. double: It stores 8 bytes. 7. boolean: It stores 1 bit. 8. char: It stores 2 bytes.
