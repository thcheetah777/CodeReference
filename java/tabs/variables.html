Variables

Below are a few types of variables. For every variable, you must declare it in order to use it in your code.

  1. String: A variable that stores a word or a set of words.
  2. Int: A variable that stores a integer.
  3. Char: A variable that stores a character.
  4. Boolean: A variable that stores either false or true.
  5. Float: A varible that stores decimal. Not that the value must end with a f.
  6. Double: A variable that stores fractional numbers. Note that the value must end with a d.
