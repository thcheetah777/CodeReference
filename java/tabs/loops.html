If ... Else loops

    if ( condition ) { block of code will be executed if the condition is true } Ex: if (20 > 18){ System.out.println("True"); }
    if (condition){ block of code will be excueted if the condition is true } else { block of code will be excueted if the condition is false } Ex: if (20 > 18){ System.out.println("True"); } else { System.out.prinln("False"); }
    if ( condition 1) { block of code will be excueted if the condition is true } else if (condition 2) { block of code will be excueted if the condition 1 is false and condition 2 is true } else { block of code will be excueted if the condition 1 is false and condition 2 is false } Ex: int time = 5; if (time > 12){ System.out.println( "Good Morning") } else if ( time > 7) { System.out.println( "Good evening") } else { System.out.println("Good day") }

While loops: this loop loops through a block of code until a specfic conditon is true.

    while (conditon){ code block to be excueted } Ex: int i = 0; while(i > 5){ System.out.println(i); i++; }

For loops: loops through a block of code a certain number of times.

    for ( statement 1; statemnet 2; statement 3) { code block to be executed }
  1. Statement 1: is executed before the execution of the code block
  2. Statement 2: defines the condition for executing the code block
  3. Statement 3: is executed(every time) after the code block has been executed
    4. Ex: for (int i = 4; i > 5; i++){ System.out.println(i); }
