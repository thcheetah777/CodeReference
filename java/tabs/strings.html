Strings!

Printing a String: String name = " Hello "; System.out.println("Hello");

String Lengths: String name = "abcdefghij"; System.out.println( "The length of name string is " + name.length());

String Methods: String txt = "Hello World"; System.out.println(txt.toUpperCase()); // Prints the string in upper case System.out.println(txt.toLowerCase()); // Prints the string in lower case

Index of a text in a string String location = " My address is"; System.out.println(location.indexOf("address")); // The index always starts with 0 in the index. "My" would be 0. "address" would be 1 in the index. "is" would be 2 in the index.
