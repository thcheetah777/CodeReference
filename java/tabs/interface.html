Interface:

    An interface is a completely "abstract class" that is used to group related methods. interface Animal{ public void mySound(); public void myName(); }

Example: interface Animal { public void mySound(); public void myName(); } // Pig "implements" the Animal interface class Pig implements Animal { public void mySound() { System.out.println("The pig says" + mySound); } public void myName() { System.out.println("The pig's name is" + myName); } } class MyMainClass { public static void main(String[] args) { Pig myPig = new Pig(); myPig.mySound(); myPig.myName(); } }
