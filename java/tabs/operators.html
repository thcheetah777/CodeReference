Operators!

There are 4 types of operators.

  1. 1. Arithmetic operators
  2. 2. Assignment operators
  3. 3. Comparison operators
  4. 4. Logical operators

Arithmetic operators:

  1. 1. Addition: adds two values together
  2. 2. Subtraction: subtracts one value from another
  3. 3. Multiplication: Multiplies 2 values together
  4. 4. Division: divides one value by another
  5. 5. Modulus: Returns division remainder
  6. 6. Increment: Increases value by a variable of 1
  7. 7. Decrement: Decreases value by a variable by 1

Assignment operators:

  1. 1. =: Equals the assigned value to the variable Ex: int x = 15;
  2. 2. +=: adds a value to a variable Ex: x += 5;
  3. 3. -=: subtracts a value from the variable Ex: x -= 5;
  4. 4. *= : multiplies a value to the variable Ex: x *= 2;
  5. 5. /=: divides a value by the variable Ex: x /= 2;
  6. 6. %=: gives the division remainder Ex: x %= 2;
  7. 7. ^=: to the power of the number Ex: x ^= 3;

Comparison operators

  1. 1. ==: Equal to
  2. 2. !=: not Equal
  3. 3. >: Greater than
  4. 4. <: less than
  5. 5. >=: Greater than or equal to
  6. 6. <=: less than or equal to

Logical Operator:

  1. 1. &&: both statements are true Ex: x > 5 && x > 10;
  2. 2. ||: one statement is true and the other is false Ex: x > 5 || x > 20
  3. 3. !: both statements are false Ex: !(x > 5 && x > 10);
