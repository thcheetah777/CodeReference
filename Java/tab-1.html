Variables

There are five main types of variables. 1. String: A variable in terms of a word or a set of words. For every variable, you must declare it in order to use it in your. 2. Int: a variable in terms of an integer. 3. Char: a variable in terms of a character. 4. Boolean: a variable that is either false or true. 5. Float: a varible that is a decimal. 6. double: a variable in terms of is an integer or decimal but is written as a decimal.
