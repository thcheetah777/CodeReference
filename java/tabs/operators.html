Operators!

There are 4 types of operators.

  1. Arithmetic operators
  2. Assignment operators
  3. Comparison operators

Arithmetic operators:

  • Logical operators
    1. Addition: adds two values together

    Assignment operators:

  • Subtraction: subtracts one value from another
  • Multiplication: Multiplies 2 values together
  • Division: divides one value by another
  • Modulus: Returns division remainder
  • Increment: Increases value by a variable of 1
  • Decrement: Decreases value by a variable by 1
    1. =: Equals the assigned value to the variable Ex: int x = 15;
      2. Ex: x += 5; Ex: x -= 5;
    2. +=: adds a value to a variable
      3. Ex: x *= 2;
    3. -=: subtracts a value from the variable
      4. Ex: x /= 2;
    4. *= : multiplies a value to the variable
      5. Ex: x %= 2;
    5. /=: divides a value by the variable
      6. Ex: x ^= 3;
    6. %=: gives the division remainder
  • ^=: to the power of the number

    Comparison operators

    1. ==: Equal to
    2. !=: not Equal
    3. <: less than
    4. >=: Greater than or equal to
    5. <=: less than or equal to
    6. >: Greater than

    Logical Operator:

    1. &&: both statements are true Ex: x > 5 && x > 10;
    2. ||: one statement is true and the other is false Ex: x > 5 || x > 20
    3. !: both statements are false Ex: !(x > 5 && x > 10);
